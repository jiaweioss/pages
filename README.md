# Home Vocabulary

A static front-end vocabulary page for learning common objects in different rooms.

## Features

- Room carousel for Kitchen, Drawingroom, and Bedroom
- Click a room to view common objects inside it
- English labels under every object
- Responsive layout for desktop and mobile
- No backend and no build step required

## GitHub Pages Deployment

This project can be deployed directly with GitHub Pages.

### Option 1: Deploy From Branch

1. Push this project to a GitHub repository.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Set `Source` to `Deploy from a branch`.
5. Choose branch `main` and folder `/ (root)`.
6. Save, then open the generated GitHub Pages URL.

### Option 2: Deploy With GitHub Actions

The repository also includes `.github/workflows/pages.yml`, which publishes the static files from the repository root.

## Local Preview

Open `index.html` directly in a browser.
